Giải pháp xây dựng Devops,Secops, MLops: AKS trên Azure và triển khai ứng dụng (Web, App). CI/CD Pipeline với GitHub Actions

Mô tả

Đề tài này triển khai giải pháp DevSecOps toàn diện nhằm mục đích tự động hóa việc xây dựng, kiểm thử và triển khai ứng dụng container, đồng thời tích hợp bảo mật xuyên suốt vòng đời phát triển.

Công nghệ sử dụng

  • Azure Kubernetes Service (AKS)
  • Azure Container Registry (ACR)
  • Azure Database for MySQL
  • Github Action
  • SonarCloud, Trivy
  • Ingress Controller, Azure Load Balancer

Kết quả

Đề tài "Giải pháp Xây dựng DevSecOps Toàn diện" không chỉ dừng lại ở việc triển khai ứng dụng mà còn thiết lập một quy trình làm việc hiện đại, giải quyết các thách thức về tốc độ, chất lượng và bảo mật trong phát triển phần mềm.
